# Vampire Survivors Clone

## Project Description
A clone of the popular game Vampire Survivors.

## Setup and Run
1. Clone the repository.
2. Open the project in Unity Hub.
3. Select the correct Unity Editor version (as specified in ProjectSettings/ProjectVersion.txt).
4. Open the MainMenu scene (Assets/Scenes/MainMenu.unity).
5. Click the Play button in the Unity Editor.

## Main Features
- Endless wave-based survival.
- Multiple playable characters with unique starting weapons.
- Variety of weapons and upgrades.
- Different enemy types.
- Experience and leveling system.
- Coin collection and persistent upgrades.
